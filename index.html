Headers tag

HTML (Hypertext Markup Language) is the standard language for creating web pages and applications.

Syntax Code


<h1>Hello World</h1>
<h2>Hello World</h2>
<h3>Hello World</h3>
<h4>Hello World</h4>
<h5>Hello World</h5>
<h6>Hello World</h6>

Output

Hello World

Hello World

Hello World

Hello World

Hello World
Hello World

Quotation and Citation

HTML (Hypertext Markup Language) is the standard language for creating web pages and applications.

Syntax Code


<b>bold test</b>
<strong>strong test</strong>
<italic>italic test</italic>
<em>empharize test</em>
<mark>marked test</mark>
<small>small test</small>
<del>deleted test</del>
<ins>small test</ins>
<h1>5<sup>2</sup>=25</h1>
<h1>H<sub>2</sub>O</h1>

Output

bold test
strong test
italic test
empharize test
marked test
small test
deleted test
small test

52=25
